sac** miniseed_trace_list_to_sac ( MS3TraceList *  mst3k)

Convert a Miniseed Trace List to a set of sac files.

Convert a Miniseed Trace List into a set of sac files. Everything possible is pulled from the miniseed data file including

  • Station Network, Name, Component, and Location
  • Data start and end times
  • Filename: Net.Sta.Loc.Cha.Qual.Year.Day.HMS.sac
