Fill meta data associated with an event in multiple sac file.

Parameters
ssac files, encolsed in an xarray
evevent data, if NULL, nothing is done
verbosebe verbose when setting meta data
Note
Fields set are
  • evla - Event Latitude
  • evla - Event Longitude
  • evla - Event Depth
  • evla - Event Elevation - Always 0.0
  • kevnm - Event ID - Only if <= 16 characters
  • kzdate and kztime - Event Origin time
  • iztype = IO (Origin)
All time picks are shifted so the origin time is 0.0, See sac_set_time()
